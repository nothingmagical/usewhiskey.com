# Whiskey

The website for [Whiskey](https://github.com/nothingmagical/whiskey) and [Whiskey for iOS](https://github.com/nothingmagical/whiskey-ios).

## Running Locally

    $ git clone https://github.com/nothingmagical/usewhiskey.com.git
    $ cd usewhiskey.com
    $ bundle install
    $ bundle exec shotgun

Enjoy.
